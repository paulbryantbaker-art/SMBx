Build notes for Claude Code

Color ruleUI chrome is strictly greyscale. Black for primary buttons, active states, hero typography. Content can carry color — fit scores, status dots, document thumbnails. No terra, no brand color in chrome anywhere.
Logo treatmentPlain text only: smbx.ai or X. No color on any letter. Inherits the surrounding text color. Sora 800 weight.
Glass recipeTab bar: rgba(255,255,255,0.72) + blur(32px) saturate(1.8). Mini bar: rgba(255,255,255,0.78) + blur(30px) saturate(1.8). Both get inset 0 0.5px 0 rgba(255,255,255,0.9) for the specular highlight and 0.5px solid rgba(0,0,0,0.08) hairline borders. Test on iOS Safari — needs -webkit-backdrop-filter.
Yulia avatarDark charcoal-to-black gradient: linear-gradient(135deg, #3A3A3E, #0A0A0B) with white Y and inset highlight. This is the one piece of "character" in an otherwise monochrome chrome — she feels like a piece of hardware or polished graphite.
Status indicators7px colored dots, not colored pills. Green = ready, amber = in progress, red = flag, grey = draft. The dot carries the signal; the text stays grey/black.
Content-level colorAllowed: score band pills (green/amber/grey), flag counts (red 991B1B on red-50 FEE2E2), chart fills (dark gradient), document thumbnails (whatever the doc contains). Not allowed: UI accents, brand highlights, "it's a link" color.
Component hierarchyAppShell → (TopBar, TabContent, YuliaAgent, TabBar). YuliaAgent owns full/mini/side state. Tabs are lazy-loaded routes: /deal, /docs, /pipeline, /search. Canvas tabs live inside /docs as a second-level routing dimension.
Safe areaTab bar height 78px absorbs env(safe-area-inset-bottom). Scroll area gets padding-bottom: 148px (tab + mini + gap). Use visualViewport for keyboard handling.
Progressive shell revealFrom public morph: full chat first, no shell. After Yulia's first deliverable, collapse chat to mini and frost the tab bar in from below. Pipeline stays dim until sourcing data exists.
Out of scopeInvitations/sharing, notifications, settings, dark mode variant. Dark mode is a straightforward inversion of this palette — design it after light mode ships.
