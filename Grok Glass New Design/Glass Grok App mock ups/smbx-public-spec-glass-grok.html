The problem

75% of owners who sell their business regret it within a year.

The number comes from the Exit Planning Institute. Thousands of former owners surveyed. The regrets are almost always the same.
They weren't financially prepared. They left hundreds of thousands — sometimes millions — on the table. In add-backs they never identified. In tax structures they never modeled. In competitive processes they never ran.
Hero 1 · Add-backs

The money hiding in your tax returns.

Reported EBITDA and real EBITDA are almost never the same number.
Your accountant's job is to minimize your taxes. A buyer's job is to maximize the price they'll justify to their lender or investment committee. Somewhere between those two numbers is the truth — and the gap is almost always larger than you think.

Journey page system · applies to /sell, /buy, /raise, /integrate
Common hero structure: eyebrow · H1 · tag · chat input · chips. Common section rhythm: alternate bg:var(--bg-app) with bg:var(--bg-card) to create banding without color. "Hero" sections (the 3 capability anchors per page) use the "tint" class. Dark-inverted sections — like the SBA alert banner on /buy — use bg:var(--accent) with white text, borrowing the pattern from the Blind Equity callout in the app spec. Bottom CTA on every journey page re-uses the hero chat input pattern with a shorter tagline.

Build notes for Claude Code · Glass Grok

Journey page template/sell, /buy, /raise, /integrate all use the same hero structure: eyebrow + H1 + tag + chat input + chips. Only copy changes. Build once as a <JourneyHero> component, pass copy as props. Same for bottom CTA — reuse the hero chat pattern.
Glass nav pillFloating absolute; top:42px; left:12px; right:12px. rgba(255,255,255,0.88) + blur(28px) saturate(1.8) + hairline border + inset top highlight. Sticky via intersection observer — morphs to a smaller pill on scroll. "Start free" CTA is a black pill; logo is plain text smbx.ai.
Section rhythmAlternate background:var(--bg-app) with background:var(--bg-card) to create banding. Dark-inverted sections (background:var(--accent); color:#fff) reserved for SBA alert banner on /buy and any "Blind Equity / Yulia's take" callouts.
Logo everywherePlain text only: smbx.ai or X. No colored letter. Sora 800, inherits surrounding text color. No emoji, no icon variant, no SVG logo. The wordmark is the logo.
Chat input = conversion elementOn every page. Same recipe: white card, 20px radius, inset top highlight, soft shadow, plus button + field + black send circle. Placeholder text is page-specific. On Home it rotates every 4 seconds through 6 hints.
Interactive calculatorsThree on the site: add-back estimator (/sell), deal rundown preview (/buy), sell-vs-raise calc (/raise), Day 1 checklist generator (/integrate). All render inline, progressive disclosure (input appears → result streams in → CTA to continue in chat).
Color rule holdsContent-level color only: score band pills (green/amber/grey), SBA alert banner (dark-inverted, no red), stat numbers (dark typography). Chrome stays greyscale. No brand color replaces terra.
Morph to appCSS state machine: landing → chat → app. Public nav fades out, topbar + tab bar + mini bar fade in. Same AppShell component as internal spec. URL updates via pushState, not route change. Journey context captured to session at first message.
TypographyH1: Sora 800 / 34–42px / -0.03em. H2 (section headings): Sora 800 / 26px / -0.025em. Eyebrow: Sora 700 / 10px / uppercase / 0.14em tracking / grey. Body: Inter 500 / 13.5px / line-height 1.65. Lead body: Inter 500 / 15px / primary text color.
Pricing table mobileAt <320px it becomes a vertical stack of tier cards, not a table. Each card: tier name + price + key differences. Full comparison available as a toggle/expand. Enterprise card links to /enterprise.
