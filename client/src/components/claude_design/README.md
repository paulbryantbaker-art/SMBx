# Archived Claude Design Handoffs

This folder contains historical Claude Design handoff artifacts. It is excluded
from TypeScript compilation and is not a production source of truth.

Do not build new UI or pricing copy from these files. Current implementation
source is the V6 app shell and the root design handoff bundles named in
`AGENTS.md`, `CLAUDE.md`, `DESIGN_SOURCE.md`, and `REPO_STATUS.md`.
